# Smart Fridge Manager V3

GitHub-Pages-kompatible HTML-/CSS-/JavaScript-App.

## Funktionen

- modernes Dark-Dashboard
- realistische Kühlschrankgrafik
- Produkte direkt im Kühlschrankfach
- Zonen A bis F
- Produktdetail-Popup
- Barcode-Scan per Handy-Kamera, Live-Kamera oder Eingabe
- OpenFoodFacts-Anbindung
- Produktbilder aus OpenFoodFacts
- Bestand mit + / − und Entnehmen
- Einkaufsliste
- Rezeptvorschläge
- Statistik
- mobile Navigation

## GitHub Pages

Diese Dateien und Ordner ins Repository hochladen:

- index.html
- css/
- js/
- README.md
