# Kühlschrank Manager

Reine HTML-/CSS-/JavaScript-App für GitHub Pages.

## Start lokal

Die Datei `index.html` per Doppelklick öffnen.

## GitHub Pages

1. Neues GitHub Repository erstellen, z.B. `kuehlschrank-manager`
2. Diese Dateien hochladen:
   - `index.html`
   - `style.css`
   - `app.js`
   - `README.md`
3. In GitHub auf **Settings**
4. Links auf **Pages**
5. Bei **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Speichern

Danach ist die App über GitHub Pages erreichbar.

## Funktionen

- Dashboard
- Lebensmittel eintragen
- Ablaufdatum überwachen
- Fast-leer-Anzeige
- Einkaufsliste
- Rezeptvorschläge
- Statistik
- Dark Mode
- Speicherung lokal im Browser per LocalStorage

## Hinweis

Die Daten werden lokal im Browser gespeichert. Wenn der Browser-Cache gelöscht wird, sind die Daten weg.
